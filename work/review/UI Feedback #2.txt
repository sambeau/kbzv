# UI feedback #2

Here’s the second round os feedback of the App UI. 
Most of the major issues have been fixed, so lets Strat honing on on the medium-sized issues.

## Filters
- The filters on Docs work in the opposite way to the filters on Work.
- Docs is better—nothing ticked means *‘all’*. 
- Can you apply the same pattern to Work
	- nothing ticked means *‘all’*
	- A tick means only
	- Multiple ticks means only(A and B and C)
